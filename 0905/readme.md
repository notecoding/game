# 🎮 게임 프로그래밍 OT 정리

## 📌 게임 프로그래밍 전반적인 흐름 이해
- 게임 루프(Game Loop) 구조 이해  
  - **입력(Input)** → **업데이트(Update)** → **렌더(Render)** 반복  
- 메모리 구조 및 자원 관리 개념  
- 스프라이트, 충돌 처리 등 기본 시스템의 동작 원리 소개  
- 엔진 없이 **C 언어 기반으로 직접 시스템 구현하는 흐름** 이해  
- 실습 및 프로젝트 진행 방식을 안내받음  

---

## 🛠 사용 언어 및 개발 환경
- **Language**: C  
- **Compiler**: gcc / clang  
- **Tools**: Visual Studio Code / CLion / CodeBlocks  
- **Version Control**: Git, GitHub

---

## 🎯 향후 학습 목표
- C 언어로 게임 루프 직접 구현  
- 키보드 입력 처리  
- 화면 출력 및 간단한 그래픽 표현  
- 충돌 판정 및 객체 관리  
- 간단한 미니 게임 제작  
- 팀 프로젝트를 통한 협업 경험
