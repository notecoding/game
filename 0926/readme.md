## 1. 🔢 숫자의 변환과 표현

### ✔ 문자열로 처리 (간단)
- 숫자를 문자열로 입력받아  
  자리수별로 char 배열에서 하나씩 출력  
- 높은 단위 → 정방향 출력  
- 낮은 단위 → 역순 출력

### ✔ 정수로 처리 (연산 활용)
- 나누기(/), 나머지(%) 사용  
- 높은 단위 출력 → `log10()` + `pow()`로 자리수 계산  
- 낮은 단위 출력 → `number % 10` 반복

### ✔ 재귀 활용
- 상위 자리수를 나눠가며 재귀 호출  
- `serial_number()` → 높은 자리부터  
- `reverse_number()` → 낮은 자리부터

## 2. 🟦 디지털 숫자 출력 (LED 숫자 표현)

### ✔ 0~9 숫자를 5×4 형태로 배열에 저장
- 숫자 하나 = 20개 요소(1 또는 0)
- 1 → ■ 출력  
- 0 → 공백 출력

### ✔ 구현 구조
- `digit_print()` : 한 줄(line) 출력  
- `number_check()` : 숫자의 각 자리 탐색(재귀)  
- main에서 line 0~4 반복 → LED 형태 완성

출력 예:

■ ■■ ■
■■■■ ■■ ■■■■
■ ■■ ■
■■■■ ■■ ■■■■

## 3. 🎚 슬라이드 바(Slide Bar) 구현

### ✔ 기능 구성
- **수평 슬라이드바**  
- **수직 슬라이드바**
- 입력값으로 bar 길이 설정  
- 화살표 키로 bar 이동  
- 현재 위치 숫자 출력

### ✔ 주요 함수
- `draw_rectangle()` : 기본 사각형 출력  
- `draw_horizontal_slide()` : 가로 슬라이드  
- `draw_vertical_slide()` : 세로 슬라이드  
- `move_arrow_key()` : 이동 제어  
- `gotoxy()` : 콘솔 커서 이동

### ✔ 출력 구조
- 슬라이드 부분은 특수문자 `■` 이용  
- 방향키(←↑→↓)로 움직임  
- ESC(27) 입력 시 종료

## 4. 🔄 도형 이동 & 회전 (3×3 행렬)

### ✔ 목표
- 방향키로 도형 이동  
- 스페이스로 90도 회전  
- 콘솔 화면에서 실시간 이동/회전 구현

### ✔ 회전 원리
- 도형을 3×3 배열로 구성  
- 90도 회전 → 행렬 인덱스 변환  
temp[j][2-i] = m[i][j];

markdown
코드 복사

### ✔ 주요 구성 함수
- `rotation_right()` : 90도 시계 회전  
- `print_shape()` : 도형 출력  
- `move_shape()` : 이동 + 지연(Sleep)  
- `move_control()` : 키 입력 제어 전체 흐름  
- `gotoxy()` : 콘솔 커서 위치 변경  

### ✔ 구현 결과
- 화면에서 도형이  
- 상/하/좌/우 이동  
- space 누르면 회전  
- ESC로 종료  
- 콘솔에서 작은 ‘테트리스 도형’ 움직이는 느낌
