# 🎮 게임 프로그래밍 OT 요약 (C 언어 - 응용 모듈)

## 📌 주요 학습 내용

### 1. 🃏 트럼프 카드 구조체 활용
- 트럼프 카드 52장을 **구조체(struct)**로 표현  
  - `order` : 우선순위(♠ → ◆ → ♥ → ♣)  
  - `shape` : 카드 모양 (♠◆♥♣)  
  - `number` : 1~13 (A, J, Q, K 포함)
- 2차원 배열로 모양 저장
- 1,11,12,13은 각각 A/J/Q/K 문자 저장

### 2. 🔄 카드 섞기(Shuffle)
- 난수를 이용해 배열의 요소를 교환하여 카드 섞기
- 두 가지 방식 설명
  - **방법1:** i번째 카드와 난수 위치의 카드 교환  
  - **방법2:** 난수가 i와 같으면 재발생 → 완전한 섞기 보장

## 🎼 3. 음계(음높이) & 주파수 계산

### ✔ 음계의 기본 개념
- 라(A) = **440Hz** 기준  
- 12평균율 사용 → 반음 간 비율 약 **1.059463…**

### ✔ 음계 주파수 계산
- 옥타브마다 2배로 증가  
- 도~시까지의 주파수를 배열 및 공식으로 계산  
- 소수점 반올림하여 사용

### ✔ C에서 소리 출력
- `Beep(frequency, duration)` (Windows 전용)  
  - `<windows.h>` 필요  
- 예: "도레미파솔라시도" 재생 → 다시 역순 재생

### ✔ 숫자 키 입력으로 음 재생
- `getch()` 사용 (Enter 없이 즉시 반응)  
- '1' → 0으로 변환하기 위해 `'1' - 49` 계산

## 🔧 4. 자료 구조 개념 (게임 응용)

### ✔ 연결 리스트 (Linked List)
- 데이터 + 링크를 가진 노드 구조  
- 노드를 동적 할당해 게임의 **리플레이 기능**,  
  입력 저장 등에 사용 가능  
- ESC 입력 시 저장된 리스트를  
  순차 출력 → 역순 출력(재귀 사용)

### ✔ 스택(Stack)
- LIFO 구조  
- 새 수건 더미처럼 **항상 top에서 push/pop**  
- 카드 게임 등에서 적용 가능

### ✔ 동적 메모리
- 필요할 때만 메모리 할당  
- 사용 후 `free()`  
- 배열의 고정 크기 문제를 해결

## 🔡 5. 문자열 정렬
- 텍스트 파일에서 읽은 컴퓨터 용어 정렬  
- **대문자-소문자 ASCII 차이 때문에 직접 비교하면 오류**  
- 비교 전 소문자로 변환 → 정렬 수행  
- 원본 문자열은 그대로 유지
